# Internship Task 04: Minimalist To-Do Web Application

A clean, responsive, and highly functional To-Do List web application built as part of my Web Development Internship at SkillCraft Technology. This project focuses on DOM manipulation, event handling, and utilizing the browser's Local Storage API to persist user data.

## 🚀 Live Demo
Check out the live deployment here: [to-do-hazel-one.vercel.app](https://to-do-hazel-one.vercel.app/)

## ✨ Key Features
- **CRUD Operations:** Seamlessly Add, Read, Update (Edit), and Delete tasks.
- **Data Persistence:** Tasks are saved to the browser's `localStorage`, meaning your list stays intact even if you refresh or close the tab.
- **Smart Scheduling:** Option to attach specific dates and times to individual tasks to stay on track.
- **Dynamic State Tracking:** Easily mark tasks as complete (striking them out) or pending, with a real-time counter tracking remaining tasks.
- **Streamlined UI:** Designed with a professional, distraction-free centered card layout for the ultimate user-friendly experience.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure and form inputs.
- **CSS3:** Custom styling, Flexbox layout, and responsive mobile-first design.
- **Vanilla JavaScript (ES6+):** Application logic, array manipulation, and Local Storage integration.
